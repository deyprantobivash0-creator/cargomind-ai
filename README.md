# CargoMind AI 🚚

AI-powered customer support agent for freight forwarding and logistics companies.

## Features

- Shipment tracking
- AI-powered customer support
- Natural language understanding
- Shipment database lookup
- Gemini AI integration
- Modern chat interface
- Real-time responses

## Technology Stack

Backend:
- Python
- Flask

AI:
- Google Gemini API

Frontend:
- HTML
- CSS
- JavaScript

Database:
- JSON (current version)

## Project Architecture

## Setup

Clone repository:

```bash
git clone YOUR_GITHUB_REPO_LINK
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run:

```bash
python app.py
```

Open:

```bash
http://127.0.0.1:5000
```
